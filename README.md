# Power Factor Correction System (ESP32)

## Overview

IoT-based system for real-time power factor monitoring and automatic correction using ESP32 and capacitor bank switching.

## Features

* Real-time voltage, current, PF monitoring
* Automatic capacitor switching
* No-load detection (PF = 1 logic)
* LCD display
* Blynk IoT monitoring

## Tech Stack

ESP32 | Embedded C | PZEM004T | Blynk | Relay Control

## Note

Full implementation includes IoT monitoring (Blynk), LCD display, and advanced control logic which are not included in this public version.
